# 인형뽑기 (Claw Machine) — 스펙 (SDD)

바이브 코딩의 기준 문서. 각 요구사항은 ID를 가지며, **Phase 완료 = 해당 수용 기준(AC) 전부 통과**.
검증 방식: `[T]` = vitest 단위 테스트, `[C]` = 브라우저 플레이스루 체크리스트.

## R1. 기계와 인형 구덩이

- [x] **R1.1** `[C]` 캐비닛(바닥·4벽·배출구 개구부)이 렌더되고, 인형이 벽을 뚫고 나갈 수 없다
- [x] **R1.2** `[T][C]` 카탈로그의 모든 인형 종류(≥4종)가 서로 다른 외형으로 렌더된다
- [x] **R1.3** `[C]` 인형 20개 낙하 후 5초 내 더미가 안정(슬립)되고, 지터가 없다 (60fps 근처 유지)

## R2. 클로 조작 (정통 2버튼)

- [x] **R2.1** `[T][C]` 버튼①을 누르는 동안 클로가 +X로 이동, 릴리스 시 정지 (경계 클램프). 재홀드 허용(관대 정책 — 여러 번 눌러도 됨)
- [x] **R2.2** `[T][C]` 이후 버튼②를 누르는 동안 +Z로 이동, 릴리스 시 **자동 하강** 시작. 재홀드 허용
- [x] **R2.3** `[T][C]` 클로는 최저점에서 손가락을 닫고 상승→원위치 복귀→개방까지 자동 진행
- [x] **R2.4** `[C]` 키보드와 화면 터치 버튼이 동일하게 동작한다 (모바일 뷰포트 포함) — 키보드 확인됨. 모바일 뷰포트 실기기 확인은 Phase 6
- [x] **R2.5** `[T]` 라운드 시작 후 `roundTimeLimit`(20초) 초과 시 버튼②를 누르지 않아도 자동으로 하강 단계로 전이한다 (테스트로 검증, `stateMachine.test.ts`)

## R3. 잡기 판정

- [x] **R3.1** `[T][C]` 손가락이 닫힐 때 손바닥 기준 반경(`grabRadius`) 내 최근접 인형 1개가 후보가 된다 — 순수 함수 `pickNearest`(`logic/grab.ts`), 반경 밖/빈 배열은 null
- [x] **R3.2** `[T]` 성공 확률은 `gripStrength`를 따른다 (RNG 주입식 — `logic/grab.ts`)
- [x] **R3.3** `[C]` 성공 시 인형이 조인트로 매달려 흔들리며 운반된다 (LIFTING까지 실동작 확인 — 흰 토끼가 클로와 함께 들림)
- [x] **R3.4** `[T]` 확률적 "중간 낙하"가 발생할 수 있다 (시점 결정 순수 함수 — `logic/grab.ts`)
- [~] **R3.5** `[C]` 배출구 상공에서 릴리스된 인형이 배출구 센서를 통과하면 승리로 기록된다 — 해당 인형은 제거되고, 카탈로그의 `score`만큼 누적 점수에 가산된다. **RESULT까지 전체 라운드 완주는 브라우저 자동화 리로드로 중간 끊김 — Phase 4 HUD 토스트 작업과 함께 재확인 필요**

## R4. 게임 루프·HUD

- [x] **R4.1** `[C]` 크레딧 차감(시작 10, 판당 1) → 1판 진행 → 결과 토스트 → IDLE 복귀가 무한 반복 가능하다. 크레딧 0이면 새 라운드 시작 불가, HUD "코인 채우기" 버튼으로 리필 — 차감·안내 문구 전환 실동작 확인, 결과 토스트는 wonThisRound 반응형 렌더 (코드 검증 + 부분 실동작)

## 아키텍처 규칙 (구현 시 준수)

1. **`src/logic/`은 Three/Rapier 임포트 금지** — 순수 함수만, 전부 테스트 대상
2. **매직넘버는 전부 `src/config.ts`** — 튜닝 세션은 단일 파일 수정
3. 클로는 `kinematicPosition` 바디, 이동은 반드시 `setNextKinematicTranslation()` (속도 클램프)
4. 컨테인먼트는 두꺼운(≥0.15) 투명 CuboidCollider — 얇은/트라이메시 콜라이더 금지
5. 타이머는 timestamp 기반 (`setTimeout` 금지)
6. 게임 상태: `IDLE → MOVING_X → MOVING_Z → DROPPING → GRABBING → LIFTING → RETURNING → RELEASING → RESULT → IDLE`
7. 라이선스: 의존성·에셋은 MIT/Apache-2.0/CC0만
